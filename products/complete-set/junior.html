Cleveland Golf's Junior Sets make it easy for youngsters to grow into a game they can enjoy for a lifetime. Each junior club features similar technology found in Cleveland Golf's standard equipment.

OPTIMIZED SETS:

Cleveland Golf's Junior Sets are available in three different options, with each club optimized for juniors in specific age and size ranges. Carry Bag is included.

LARGE SET:

7-piece set suits juniors age 10-12 or 54" - 63" tall.
Includes: Driver (18°), Fairway (22°), Hybrid (28°), 7 & 9 Iron, Wedge (56°), Putter

MEDIUM SET:

6-piece set suits juniors age 7-9 or 44" - 53" tall.
Includes: Driver (18°), Hybrid (28°), 7 & 9 Iron, Wedge (56°), Putter

SMALL SET:

3-piece set suits juniors age 4-6 or 36" - 43" tall.
Includes: Fairway (22°), 7 Iron, Putter
